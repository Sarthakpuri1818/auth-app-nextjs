# Full-stack authentication system built with:

- Next.js (App Router)
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt
- Protected Routes
- Login / Signup / Logout

- User Registration
- Secure Login
- JWT Cookie Authentication
- Logout Functionality
- Protected Profile Route

## Tech Stack

Next.js | MongoDB | TypeScript | Axios
