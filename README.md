

# Auth App - Next.js

# Full-stack authentication system built with:

- Next.js (App Router)
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt
- Protected Routes
- Login / Signup / Logout

## Features

- User Registration
- Secure Login
- JWT Cookie Authentication
- Logout Functionality
