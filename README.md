

# Auth App - Next.js

# Full-stack authentication system built with:

- Next.js (App Router)
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt
- Protected Routes
- Login / Signup / Logout

## Features

- User Registration
- Secure Login
- JWT Cookie Authentication
- Logout Functionality


- Protected Profile Route

## Tech Stack

Next.js | MongoDB | TypeScript | Axios

## Demo Flow

1. Register a new account
2. Login with credentials
3. Access protected profile route
4. Logout securely
